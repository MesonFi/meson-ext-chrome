# Design Guide

## Color Palette

| Name          | Hex      | Usage                              |
| ------------- | -------- | ---------------------------------- |
| white         | #FFFFFF  | page background, card background   |
| black         | #000000  | primary text                       |
| primary       | #25C696  | primary buttons, interaction color |
| primary-hover | #1BE2BB  | primary button hover state         |
| secondary     | #6F9483  | secondary text, icons              |
| color-strong  | #25372E  | emphasized text                    |
| color-regular | #75807B  | regular text                       |
| color-muted   | #BDC7C3  | muted hints                        |
| border        | #DDF3EA  | borders, dividers                  |
| surface       | #F2F9F7  | card and module backgrounds        |
| disabled      | #DFEBE7  | disabled state backgrounds         |
| success       | #11C58F  | success states and messages        |
| warning       | #FFC700  | warning states and messages        |
| error         | #FD5050  | error states and messages          |
