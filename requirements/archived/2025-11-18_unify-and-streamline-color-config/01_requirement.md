# 需求

在项目的 `tailwind.config.js` 及相关 CSS 文件中，当前定义了大量未使用或近似重复的颜色，导致颜色表臃肿，不易维护。我们需要简化并统一颜色定义，以提高可维护性和扩展性。

## 要求

1. 分析现有代码和组件中实际使用到的颜色，识别未使用的颜色并移除。  
2. 对于相近或重复的颜色值，合并为一致的新颜色，可允许略有差异以优化视觉体验。  
3. 在 `requirements/02_design.md` 中提供最终的新颜色表和实现方案，包括：  
   - 每个新颜色的名称与十六进制色值；  
   - 所有旧颜色名称及其色值到新颜色名称的映射关系；  
   - 针对 `tailwind.config.js` 及相关 CSS 文件的修改步骤和示例。  

请不要在 `02_design.md` 中描述如何调配新颜色，仅提供最终确定的颜色表及配套的更新方案。
